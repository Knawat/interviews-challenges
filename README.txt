== Description ==



Kanawat Plugin is a plugin that is used to enable you to use show your products randomly using the shortcode [my_random_products]


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Download the plugin and save it under wordpress\wp-content\plugins\
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

How can i change the number of the products?

1- Go to wordpress dashboard 
2- Plugin editor
3- On the upper right choose Kanwat Plugin, then press select
4- Go to line 64, change 'posts_per_page' to the number you want.
5- Press update file
